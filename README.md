# Kraker

Kraker is a genetic algortihm to find a text.

## Demo

https://replit.com/@sultandlen/kraker

## Run Locally

Clone the project

```bash
  git clone https://github.com/sultandlen/kraker.git
```

Go to the project directory

```bash
  cd kraker
```

Compile

```bash
  javac Kraker.java
```

Run

```bash
  java Kraker
```
